# Ícones PWA

Para o PWA funcionar completamente, coloque aqui:

- **icon-192.png** (192x192 pixels)
- **icon-512.png** (512x512 pixels)

Você pode criar ícones personalizados com o logo do VEO Automator usando ferramentas como:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/

Por enquanto, o PWA funcionará sem ícones, mas ao instalar no celular, usará um ícone padrão.
